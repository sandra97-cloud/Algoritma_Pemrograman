4. Operator Aritmatika dalam Java

Operator Aritmatika Binary

Simbol Nama Deskripsi
* Perkalian Melakukan perkalian dua operand.
+ Penjumlahan Menjumlahkan dua operand.
- Pengurangan Mengurangi suatu operand.
/ Pembagian Membagi suatu operand.
% Modulus Menghasilkan sisa hasil pembagian.

Operator Aritmatika Unary

Simbol Nama Deskripsi
+ Positif Menandakan nilai positif.
- Negatif Menandakan nilai negatif.
++ Increment Menambah nilai variabel sebesar 1.
-- Decrement Mengurangi nilai variabel sebesar 1.
