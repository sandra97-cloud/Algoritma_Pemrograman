1. Apa yang Anda ketahui tentang instruksi pada algoritma?

Instruksi dalam algoritma merupakan serangkaian perintah yang disusun secara terstruktur untuk mencapai tujuan tertentu. Instruksi ini menjadi panduan langkah demi langkah dalam menyelesaikan masalah sehingga harus runtut, logis, dan tidak membingungkan.

2. Apa yang dimaksud dengan struktur algoritma sekuensial?

Struktur algoritma sekuensial adalah bentuk algoritma dimana proses dijalankan secara linear atau berurutan tanpa adanya pemilihan kondisi maupun pengulangan. Semua langkah dilakukan satu per satu sesuai urutan penulisan.

3. Karakteristik Struktur Sekuensial

No Karakteristik
1 Alur program berjalan lurus (linear).
2 Tidak ada kondisi untuk memilih langkah tertentu.
3 Tidak ada proses pengulangan.
4 Urutan langkah sangat berpengaruh pada hasil.
5 Mudah dibuat dan dipahami.

4. Contoh Sederhana Penggunaan Struktur Sekuensial

Studi Kasus: Menghitung Nilai Rata-rata

Algoritma

  1. Mulai
  2. Masukkan nilai pertama
  3. Masukkan nilai kedua
  4. Hitung rata-rata = (nilai1 + nilai2) / 2
  5. Tampilkan hasil rata-rata
  6. Selesai

Pseudocode

Mulai Input nilai1 Input nilai2 rata = (nilai1 + nilai2) / 2 Output rata Selesai
