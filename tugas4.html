Studi Kasus

Seorang guru ingin menentukan status kelulusan siswanya berdasarkan nilai ujian mata pelajaran Matematika. Jika nilai siswa lebih besar atau sama dengan 75, maka siswa dinyatakan "LULUS". Jika nilai di bawah 75 maka dinyatakan "TIDAK LULUS".

Notasi Kalimat Deskriptif

Algoritma_CekKelulusan

Deklarasi

Variabel Tipe Data
nama_siswa String
nilai_ujian Integer
keterangan String

Deskripsi Algoritma

  1. Masukkan nama siswa.
  2. Masukkan nilai ujian siswa.
  3. Lakukan pengecekan kondisi.
  4. Jika nilai ujian ≥ 75 maka keterangan = "LULUS".
  5. Jika nilai ujian < 75 maka keterangan = "TIDAK LULUS".
  6. Tampilkan nama siswa dan status kelulusannya.
  7. Selesai.

Notasi Pseudocode

PROGRAM CekKelulusanUjian DEKLARASI nama_siswa : string nilai_ujian : integer keterangan : string ALGORITMA read(nama_siswa) read(nilai_ujian) IF nilai_ujian >= 75 THEN keterangan = "LULUS" ELSE keterangan = "TIDAK LULUS" ENDIF write("Nama : ", nama_siswa) write("Status : ", keterangan) END
