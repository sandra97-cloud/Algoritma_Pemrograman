Program Perulangan For

for i in range(1,6):
    print("Perulangan ke-", i)

Output:

Perulangan ke-1
Perulangan ke-2
Perulangan ke-3
Perulangan ke-4
Perulangan ke-5

Perulangan for digunakan ketika jumlah pengulangan sudah diketahui. Pada program di atas, perulangan dilakukan sebanyak 5 kali.

Program Perulangan While

i = 1

while i <= 5:
    print("Perulangan ke-", i)
    i += 1

Output:

Perulangan ke-1
Perulangan ke-2
Perulangan ke-3
Perulangan ke-4
Perulangan ke-5

Perulangan while akan terus berjalan selama kondisi bernilai benar.

Program Do While

i = 1

while True:
    print("Perulangan ke-", i)
    i += 1

    if i > 5:
        break

Output:

Perulangan ke-1
Perulangan ke-2
Perulangan ke-3
Perulangan ke-4
Perulangan ke-5

Python tidak memiliki struktur do while secara langsung. Oleh karena itu digunakan while True dan break untuk meniru perilaku do while.

Program Perulangan Bersarang (Nested Loop)

for i in range(1,6):
    for j in range(i):
        print("*", end=" ")
    print()

Output:

*
* *
* * *
* * * *
* * * * *

Perulangan bersarang adalah perulangan yang berada di dalam perulangan lainnya. Contohnya digunakan untuk membuat pola bintang seperti di atas.

Kesimpulan

Dalam bahasa Python terdapat berbagai jenis perulangan yang dapat digunakan untuk mengulang suatu proses. Perulangan for digunakan ketika jumlah pengulangan sudah diketahui, while digunakan berdasarkan kondisi tertentu, do while dapat disimulasikan dengan while True dan break, sedangkan perulangan bersarang digunakan ketika terdapat perulangan di dalam perulangan lainnya.
