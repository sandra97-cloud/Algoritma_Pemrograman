1. Apa yang dimaksud dengan Array?

Array adalah struktur data yang digunakan untuk menyimpan sekumpulan data yang memiliki tipe yang sama dalam satu variabel. Setiap data dalam array memiliki posisi yang disebut indeks, sehingga data dapat diakses dengan mudah menggunakan indeks tersebut.

Contoh:

Nilai = {80, 85, 90, 95}

2. Jelaskan yang dimaksud dengan Array 1 Dimensi!

Array satu dimensi adalah array yang hanya memiliki satu deretan elemen dan menggunakan satu indeks untuk mengakses setiap elemennya.

Contoh C++:

int nilai[5] = {70, 75, 80, 85, 90};

Keterangan:

nilai[0] = 70
nilai[1] = 75
nilai[2] = 80
nilai[3] = 85
nilai[4] = 90

3. Bagaimana Cara Mendeklarasikan Array?

Array dideklarasikan dengan menuliskan tipe data, nama array, dan jumlah elemen yang akan disimpan.

Contoh C++:

int angka[10];

Artinya:

  • int = tipe data integer
  • angka = nama array
  • [10] = jumlah elemen yang dapat disimpan sebanyak 10 data

4. Tipe Data yang Digunakan untuk Mendeklarasikan Array

Beberapa tipe data yang dapat digunakan antara lain:

Integer (int)
int angka[5];

Float
float nilai[5];

Double
double data[5];

Character (char)
char huruf[10];

String
string nama[5];

Boolean (bool)
bool status[5];

5. Macam-Macam Array

a. Array Satu Dimensi (1D)

int nilai[5];

b. Array Dua Dimensi (2D)

int matriks[3][3];

c. Array Tiga Dimensi (3D)

int data[2][3][4];

d. Array Multidimensi

Array yang memiliki lebih dari dua dimensi dan digunakan untuk menyimpan data yang lebih kompleks.
