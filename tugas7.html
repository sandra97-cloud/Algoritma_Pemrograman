1. Apa yang kamu ketahui tentang Array Dua Dimensi?

Array dua dimensi adalah array yang memiliki dua indeks, yaitu indeks baris dan indeks kolom. Array ini biasanya digunakan untuk menyimpan data dalam bentuk tabel atau matriks.

Contoh:

int nilai[2][3] = {
    {80, 85, 90},
    {75, 88, 92}
};

2. Bagaimana Cara Mendeklarasikan Array Dua Dimensi?

Array dua dimensi dideklarasikan dengan menuliskan tipe data, nama array, jumlah baris, dan jumlah kolom. 

int matriks[3][4];

Keterangan:

  • int = tipe data
  • matriks = nama array
  • 3 = jumlah baris
  • 4 = jumlah kolom

3. Selain Array Dua Dimensi, Sebutkan Macam-Macam Array!

  • Array Satu Dimensi (1D)
  • Array Dua Dimensi (2D)
  • Array Tiga Dimensi (3D)
  • Array Multidimensi
Array 1D  : int nilai[5];
Array 2D  : int matriks[3][3];
Array 3D  : int data[2][3][4];
Array Multi Dimensi : int data[2][3][4][5];

4. Apa Perbedaan Array Satu Dimensi dengan Array Dua Dimensi?

Array Satu Dimensi Array Dua Dimensi
Memiliki satu indeks. Memiliki dua indeks (baris dan kolom).
Bentuknya seperti daftar data. Bentuknya seperti tabel atau matriks.
Contoh: nilai[5] Contoh: nilai[3][4]

5. Apa Perbedaan Array Dua Dimensi dengan Array Multidimensi?

Array Dua Dimensi Array Multidimensi
Memiliki dua indeks. Memiliki lebih dari dua indeks.
Digunakan untuk data berbentuk tabel atau matriks. Digunakan untuk data yang lebih kompleks.
Contoh: int data[3][4]; Contoh: int data[2][3][4];

Kesimpulan

Array dua dimensi merupakan struktur data yang memiliki baris dan kolom sehingga cocok digunakan untuk menyimpan data berbentuk tabel atau matriks. Selain array dua dimensi, terdapat array satu dimensi, tiga dimensi, dan multidimensi yang digunakan sesuai kebutuhan penyimpanan data.
